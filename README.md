## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:
* Você instalou a versão mais recente do navegador que utiliza.
* Você tem uma máquina `<Windows / Linux / Mac>`. 
* Você possui `<VSCode>`.
